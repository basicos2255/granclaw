# GranClaw UI

Frontend para GranClaw.

## Stack

- React + TypeScript
- Vite
- Router simple (sin librería externa)
- Fetch nativo

## Scripts

```bash
npm run dev      # Desarrollo
npm run build    # Build producción
npm run preview  # Preview build
```

## Páginas

| Ruta | Descripción |
|------|-------------|
| / | Dashboard |
| /agents | Lista de agentes |
| /sessions | Lista de sesiones |
| /tasks | Lista de tareas |
| /presets | Lista de presets |
| /config | Configuración |

## Conexión

- Backend: http://localhost:3001
- Puerto UI: 5173

## Estado

- UI básica funcional
- Conectada a backend local
- Sin diseño elaborado
- Sin conexión OpenClaw
